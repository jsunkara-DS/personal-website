### Styling
The website uses a combination of TailwindCSS and custom CSS:
- **Colors**: Purple gradient theme (`#667eea` to `#764ba2`)
- **Fonts**: Inter font family
- **Animations**: CSS animations and JavaScript-powered effects
- **Dark Mode**: Persistent theme switching

### JavaScript Architecture
The codebase uses ES6+ classes for better organization:
- `TypingEffect` - Hero section typing animation
- `DarkModeManager` - Theme switching with localStorage
- `MobileMenu` - Responsive navigation
- `SmoothScrolling` - Navigation scroll behavior
- `SkillBars` - Animated skill progress
- `ContactForm` - Form validation and submission
- `NotificationManager` - Toast notifications
